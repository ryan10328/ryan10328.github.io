環境

  • Windows 10 企業版 (1709)

安裝步驟

  1. MongoDB download center

    • 選擇 Community Server 進行安裝

  2. 安裝方式就是一直下一步，就不特別截圖了

  3. 安裝完畢之後記得將 C:\Program Files\MongoDB\Server\[DB_VERSION]\bin 加入環境變數

  4. 開啟命令提示字元(建議使用最高管理者權限)，先在以下路徑建立資料夾，屆時 MongoDB 才不會在開啟的時候報錯

啟動

啟動大致上有兩種方式

  1. 直接呼叫 mongod.exe
  2. 透過 mongod.exe 註冊服務到 windows service 讓他自己開機時執行
